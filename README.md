# ai-tools-en

English SEO site for AI tool reviews. Targets US/EU AdSense (CPM 5-10x higher than zh-Hant).

- Stack: pure HTML/JS, GitHub Pages
- Writer cron: `english-seo-writer` (TBD, writes to `posts/`)
- Monetization: AdSense (apply M1) + Affiliate (placeholders in content)

## Structure
```
/
  index.html       # home
  posts/           # articles (created by cron)
  about.html       # about + affiliate disclosure
```
